# Student Placement Analytics System

## Project Overview

This project analyzes student placement data to evaluate placement performance, hiring trends, and employability factors using Excel, SQL, and Power BI.

---

## Tools Used

- Microsoft Excel
- MySQL
- Power BI

---

## Dataset

The dataset contains 500 student records with the following columns:

- Student ID
- Department
- CGPA
- Internship
- Certifications
- Company Name
- Package (LPA)
- Placement Status

---

## SQL Analysis

Performed SQL queries to analyze:

- Total Students
- Placed Students
- Placement Percentage
- Department-wise Placement
- Company-wise Hiring
- Average Package
- Highest Package
- Internship vs Placement

---

### KPI Cards
